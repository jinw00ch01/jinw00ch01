#### 🪐 [Isaac Sim 기반 화성 탐사 로버 자원 채취 시스템](https://github.com/sungyu-sung/a-2_isaac)

`ROS 2 Humble` · `Isaac Sim` · `PPO(강화학습)` · `YOLO` · `Python`

#### 🤖 [ALFRED — 지하철 자율주행 에스코트 로봇](https://github.com/sungyu-sung/rokey_alfred)
지하철역에서 승객을 목적지까지 안내하는 다중 로봇 에스코트 시스템. 시각장애인용 음성 입력, 보행 속도 맞춤 주행, 층간 릴레이 에스코트(엘리베이터 연동), 화재·부상자·위험물 감지 시 관리자 자동 알림.
`ROS 2` · `Nav2 / AMCL` · `rosbridge · WebSocket` · `React 키오스크` · `C++ · Python`
